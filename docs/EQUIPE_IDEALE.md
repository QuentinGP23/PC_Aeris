# Équipe idéale — PC Aeris

---

## Vision d'équipe

PC Aeris est un projet tech à forte composante expertise hardware. L'équipe idéale combine :

- **Compétences techniques** (développement, IA, data)
- **Passion hardware** (PC, gaming, composants)
- **Sens business** (marketing, relation client)

> Une petite équipe polyvalente et passionnée vaut mieux qu'une grande équipe cloisonnée.

---

## 1. Équipe cœur (Phase de lancement)

### 👤 Fondateur / CEO

| Aspect               | Description                                                        |
| -------------------- | ------------------------------------------------------------------ |
| **Rôle**             | Vision, stratégie, pilotage global, commercial                     |
| **Profil**           | Entrepreneur, passionné tech/gaming, à l'aise en communication     |
| **Responsabilités**  | Direction produit, partenariats, marketing, support client initial |
| **Compétences clés** | Leadership, gestion de projet, négociation, culture hardware       |
| **Temps**            | 100%                                                               |

---

### 👤 CTO / Lead Developer

| Aspect             | Description                                              |
| ------------------ | -------------------------------------------------------- |
| **Rôle**           | Architecture technique, développement, IA                |
| **Profil**         | Développeur full-stack senior, autonome, orienté produit |
| **Stack attendue** | React/TypeScript, Node.js, PostgreSQL, API REST          |
| **Bonus**          | Expérience IA/ML, intérêt pour le hardware               |
| **Temps**          | 100% (idéalement associé)                                |

**Qualités recherchées** :

- Capacité à shipper vite et itérer
- Pragmatisme (pas d'over-engineering)
- Sensibilité UX
- Autonomie forte

**Ce qu'on évite** :

- Profil "architecte" qui ne code pas
- Perfectionniste bloquant
- Aucun intérêt pour le domaine (gaming/PC)

---

### 👤 Expert Hardware / Product Owner Technique

| Aspect               | Description                                                            |
| -------------------- | ---------------------------------------------------------------------- |
| **Rôle**             | Base de données composants, règles de compatibilité, veille            |
| **Profil**           | Passionné hardware, connaissance approfondie du marché PC              |
| **Responsabilités**  | Alimenter la BDD, définir les règles IA, valider les configs           |
| **Compétences clés** | Expertise composants (CPU, GPU, RAM, etc.), benchmarks, compatibilités |

**Profil type** :

- Monteur PC expérimenté
- Rédacteur/YouTuber tech
- Ancien vendeur spécialisé (LDLC, Materiel.net)
- Membre actif de communautés hardware (HFR, Reddit PCMR)

## 4. Profil culturel recherché

### Valeurs communes

| Valeur           | Traduction concrète                                         |
| ---------------- | ----------------------------------------------------------- |
| **Passion**      | On fait ça parce qu'on aime les PC, pas juste pour l'argent |
| **Pragmatisme**  | Shipper > Perfection. Done is better than perfect.          |
| **Transparence** | Communication ouverte, pas de politique interne             |
| **Ownership**    | Chacun est responsable de son périmètre                     |
| **Curiosité**    | Veille permanente, envie d'apprendre                        |

### Red flags (ce qu'on évite)

- ❌ "C'est pas mon job"
- ❌ Ego surdimensionné
- ❌ Incapacité à recevoir du feedback
- ❌ Aucun intérêt pour le produit/domaine
- ❌ Attente d'un cadre corporate structuré

### Green flags (ce qu'on recherche)

- ✅ A déjà monté un PC
- ✅ Suit l'actu hardware (Linus Tech Tips, GN, HFR...)
- ✅ A contribué à des projets perso ou open source
- ✅ Capable de bosser en autonomie
- ✅ Communique clairement et régulièrement
